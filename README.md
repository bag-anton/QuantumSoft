# Test Task
### Python version is 3.7
## For start project:
1) create virtual environment
2) export pythonpath:
    * ```export PYTHONPATH=$(pwd)```
3) install requirements:  
    * ```pip install -r requirements.txt```
4) run web app:    
    * ```python manage.py runserver 8000 --settings=QuantumSoft.settings```
5) run pyqt app:
    * ```cd visual```
    * ```python run.py```
6) enjoy
