# Test Task
### Python version is 3.7
## For start project:
* create virtual environment
* export pythonpath:
    * ```export PYTHONPATH=$(pwd)```
* install requirements:  
    * ```pip install -r requirements.txt```
* run web app:    
    * ```python manage.py runserver 8000 --settings=QuantumSoft.settings```
* run pyqt app:
    * ```cd visual```
    * ```python run.py```
* enjoy
